# Modules: Velero (Scaffolding Placeholder)

This module directory is reserved for Velero, our disaster recovery and backup suite:
- **Scheduled Backups**: Backs up EKS cluster resources and persistent volumes (EBS).
- **Restores**: Automates system restoration to fresh clusters.
- **S3 Storage Location**: Enforces S3 bucket backup with KMS encryption.
- **IAM Policies**: IAM Roles for Service Accounts (IRSA) for Velero to write to S3 bucket.

No active resources are defined here in Phase 2. This folder serves to future-proof the platform.
